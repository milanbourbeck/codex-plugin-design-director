# Token Taxonomy

## Color Roles
- Canvas, surface, elevated surface, border, text, muted text, accent, success, warning, danger.

## Typography
- Display, heading, subheading, body, small, label, mono.

## Spacing
- Micro spacing, control spacing, card spacing, section spacing, page spacing.

## Shape
- Control radius, card radius, modal radius, full radius.

## Elevation
- Border-only, subtle shadow, overlay shadow, focus ring.

## Motion
- Duration, easing, distance, entrance, exit, state feedback.

## Breakpoints
- Compact, tablet, desktop, wide.
